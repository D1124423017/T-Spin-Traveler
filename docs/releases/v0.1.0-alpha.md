# T-Spin Traveler v0.1.0-alpha Release Notes Draft

> Draft for the first public GitHub Release. Review links, screenshots, license status, and known issues before publishing.

## Release Summary

T-Spin Traveler v0.1.0-alpha is a playable desktop browser prototype that combines Tetris-like line clearing with fantasy RPG combat.

Players use line clears, Combos, T-Spins, Back-to-Back chains, and Perfect Clears to trigger attacks, defensive feedback, upgrades, and longer-term Rift Energy progression.

## Highlights

- Tetris-like falling-block board with keyboard controls.
- RPG-style battle layer driven by line clears and advanced puzzle techniques.
- Combat feedback for T-Spins, Combos, Back-to-Back chains, Tetris clears, and Perfect Clears.
- Main menu, settings, pause, feedback, upgrade, equipment, and story-mode presentation layers.
- GitHub Pages deployment for direct browser play.
- Test suite and documentation for project structure, assets, and release checks.

## Current Scope

This alpha is intended for public playtesting and feedback. It is not a finished commercial release.

Focus areas:

- Core puzzle-to-battle loop readability.
- Desktop browser playability.
- Combat feedback clarity.
- Settings and controls usability.
- Story and UI presentation direction.
- Early project discoverability and release hygiene.

## Known Limitations

- Desktop keyboard controls are the supported target.
- Mobile touch controls are not supported.
- License choice is not finalized.
- Some presentation, equipment, and story assets are still in active iteration.
- Balance and onboarding may change based on playtest feedback.
