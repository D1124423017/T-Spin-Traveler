## English Introduction
**T-Spin Traveler** is a 2D HTML Canvas web game prototype that combines falling block puzzle gameplay with fantasy RPG combat.
Players use line clears, Combos, T-Spins, Back-to-Back chains, and Perfect Clears to trigger attacks and battle feedback.
---
## 中文介紹
**《T-Spin Traveler》** 是一款結合下落方塊消除與奇幻 RPG 戰鬥的 2D HTML Canvas 網頁遊戲原型。
玩家透過消行、Combo、T-Spin、Back-to-Back 與 Perfect Clear 觸發攻擊與戰鬥回饋。
