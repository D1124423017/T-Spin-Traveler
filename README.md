## Overview

**T-Spin Traveler** is a 2D HTML Canvas web game prototype that combines falling-block puzzle gameplay with fantasy RPG combat.

Players use line clears, Combos, T-Spins, Back-to-Back chains, and Perfect Clears to trigger attacks, defense, combat feedback, upgrades, and long-term growth.

The project is currently focused on desktop browser play with keyboard controls and static deployment through GitHub Pages.

---

## Core Gameplay

- Falling-block stacking and line clearing inspired by Tetris-like puzzle games.
- RPG-style combat triggered by player actions on the board.
- Stronger techniques such as T-Spins, Back-to-Back chains, Combos, Tetris clears, and Perfect Clears create stronger battle feedback.
- Upgrade choices and Rift Energy support longer-term progression.

## Controls

| Action | Key |
|---|---|
| Move | Arrow Left / Arrow Right |
| Soft Drop | Arrow Down |
| Hard Drop | Space |
| Rotate | Arrow Up / Z / X |
| Hold | C / Shift |
| Pause / Back | Esc |

## Visual and Audio Direction

The game uses a fantasy rift style built around ancient ruins, blue-purple magical energy, forest relics, and a mysterious grey alien traveler.

The current audio direction aims for cinematic, epic, and immersive BGM, with short responsive SFX for block handling, line clears, T-Spins, Perfect Clears, UI actions, and combat feedback.
