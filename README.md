# T-Spin Traveler

A 2D HTML Canvas game that combines falling block puzzle gameplay with fantasy RPG combat.

Players do not simply clear lines for points. Instead, they use line clears, combos, T-Spins, and stacking strategies to attack monsters, trigger skills, and control the rhythm of battle.

## 中文介紹

《T-Spin Traveler》是一款結合下落方塊消除與奇幻 RPG 戰鬥的 2D 網頁遊戲。

玩家不是單純為了消除方塊而遊玩，而是透過消除、連擊、T-Spin 與堆疊策略來攻擊怪物、觸發技能，並推進戰鬥節奏。

## Features

- Falling block puzzle gameplay
- Fantasy RPG battle system
- Line clear and combo-based attacks
- T-Spin focused gameplay concept
- Character attack animations
- Enemy HP and battle feedback
- Melee, ranged, combo, and ultimate attack effects
- HTML Canvas-based web prototype
- Playable directly in a browser

## Gameplay Concept

The core idea of this project is to connect puzzle-solving actions with battle feedback.

Clearing lines, creating combos, and performing advanced stacking techniques can trigger attacks against enemies. The goal is to make every successful clear feel impactful, turning traditional falling block gameplay into a more aggressive and satisfying battle experience.
