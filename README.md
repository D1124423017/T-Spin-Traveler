# 方塊勇者：下落方塊戰鬥遊戲

一款結合下落方塊消除與奇幻 RPG 戰鬥的 2D 網頁遊戲。  
玩家不是單純為了消除方塊而遊玩，而是透過消除、連擊與堆疊策略來攻擊怪物、觸發技能並推進戰鬥節奏。
# Block Hero: Falling Block Battle Game

A 2D web game that combines falling block puzzle gameplay with fantasy RPG combat.  
Players are not simply clearing blocks for points. Instead, they use line clears, combos, and stacking strategies to attack monsters, trigger skills, and control the rhythm of battle.
