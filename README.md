# T-Spin Traveler

**T-Spin Traveler** is a desktop browser game prototype that combines Tetris-like line clearing with fantasy RPG battle feedback.

Players stack blocks, clear lines, build Combos, chain Back-to-Back clears, land T-Spins, and trigger attacks, guard, upgrades, and Rift Energy progression.

## 10-Second Summary

- **Genre:** Tetris-like falling-block puzzle plus RPG combat.
- **Goal:** Clear lines and use advanced techniques to damage enemies and survive waves.
- **Core techniques:** T-Spins, Tetris clears, Combos, Back-to-Back chains, Perfect Clears, Hold, and Next planning.
- **Progression:** Upgrade choices, equipment, Rift Energy, and stronger battle presentation over time.
- **Platform:** PC / desktop browser, keyboard controls, HTML Canvas, GitHub Pages.

---

## Core Gameplay

| System | What it does |
|---|---|
| Falling-block board | Move, rotate, hold, soft drop, hard drop, and clear lines. |
| RPG battle layer | Puzzle actions become attacks, guard, combat feedback, and enemy pressure. |
| Advanced clears | T-Spins, Back-to-Back chains, Combos, Tetris clears, and Perfect Clears create stronger effects. |
| Upgrade flow | Between battle moments, choose upgrades that shape the run. |
| Rift theme | Ancient ruins, purple-blue rift energy, and NOA's journey through lost civilizations. |

## Controls

| Action | Key |
|---|---|
| Move | Arrow Left / Arrow Right |
| Soft Drop | Arrow Down |
| Hard Drop | Space |
| Rotate | Arrow Up / Z / X |
| Rotate 180 | A |
| Hold | C / Shift |
| Pause / Back | Esc / P |
| Music | M |

## How To Give Feedback

Feedback is welcome through GitHub Issues:

- [Bug report](https://github.com/D1124423017/T-Spin-Traveler/issues/new/choose): crashes, browser problems, controls, visual bugs, or gameplay issues.
- [Feature request](https://github.com/D1124423017/T-Spin-Traveler/issues/new/choose): new ideas, balance suggestions, UI improvements, audio feedback, or onboarding requests.
- In-game path: open **Settings -> Feedback** and use the GitHub feedback button.

When reporting a bug, include your browser, operating system, steps to reproduce, and a screenshot or video if possible.

## Visual And Audio Direction

The game uses a fantasy rift style built around ancient ruins, purple-blue magical energy, lost civilizations, and a mysterious grey alien traveler named NOA.

The current audio direction aims for cinematic BGM and responsive SFX for block handling, line clears, T-Spins, Perfect Clears, UI actions, and combat feedback.
