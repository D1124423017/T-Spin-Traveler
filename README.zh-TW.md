## 專案介紹

**《T-Spin Traveler》** 是一款結合下落方塊消除與奇幻 RPG 戰鬥的 2D HTML Canvas 網頁遊戲原型。

玩家透過消行、Combo、T-Spin、Back-to-Back 與 Perfect Clear 觸發攻擊、防禦、戰鬥回饋、升級與長期成長。

目前專案以 PC / Desktop Browser 為主，使用鍵盤操作，並透過 GitHub Pages 進行靜態部署。

---

## 核心玩法

- 以 Tetris-like 下落方塊堆疊與消行作為主要操作核心。
- 玩家在棋盤上的表現會轉化為 RPG 戰鬥行動。
- T-Spin、Back-to-Back、Combo、Tetris 消行與 Perfect Clear 會產生更強的攻擊與回饋。
- 升級選擇與 Rift Energy 提供長期成長方向。

## 視覺與音樂方向

遊戲風格以古文明遺跡、紫藍魔法裂隙、森林遺物與神秘小灰人旅者為核心。

目前音樂方向朝向電影感、宏大感、史詩感與沉浸感發展；音效則重視方塊操作、消行、T-Spin、Perfect Clear、UI 與戰鬥回饋的短促清楚反應。
